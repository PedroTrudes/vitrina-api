🚗 Vitrina API

API REST desenvolvida com Spring Boot para gerenciamento de vitrines de veículos, permitindo que lojas cadastrem, organizem e disponibilizem seus carros de forma estruturada e escalável.

A aplicação foi projetada com foco em arquitetura modular e boas práticas de desenvolvimento backend, incluindo autenticação de usuários, vinculação de lojas e sistema de convites para controle de acesso.

🔧 Tecnologias utilizadas
Java 21
Spring Boot
Spring Security
Spring Data JPA
MySQL
Maven
✨ Funcionalidades
Cadastro e autenticação de usuários
Gestão de lojas (stores)
Cadastro de contatos por loja
Sistema de convites para entrada de usuários em lojas
Estrutura preparada para integração com frontend (Angular)
🎯 Objetivo

Fornecer uma base sólida para aplicações de vitrine digital de veículos, com possibilidade de expansão para features como upload de imagens, catálogo público e integração com serviços externos.
